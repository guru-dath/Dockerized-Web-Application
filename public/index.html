Project Title: Dockerized Web Application

Developed and Dockerized a web application using Node.js and Express.js. The application provides a simple REST API to manage user data. It utilizes MongoDB as the database for storing user information. The application is containerized using Docker to ensure consistency and portability across different environments.

Key Contributions:

Created a Dockerfile to define the application's Docker image, including the installation of Node.js, copying the application code, and setting up the necessary dependencies. Configured a docker-compose.yml file to define the services needed for the application, including the web server and MongoDB database. Implemented Docker volumes to persist the MongoDB data across container restarts. Orchestrated the deployment and management of the application using Docker Compose commands, ensuring a seamless development and deployment workflow.
